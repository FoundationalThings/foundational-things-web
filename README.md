# Foundational Things

## Overview
This `foundational-things-web` repository is the current home and host of the Foundational Things website, [https://foundationalthings.com](https://foundationalthings.com).

Using GitHug Pages and a custom domain, this website serves as the web home of the tech brand of Donna J. Harris.
